✏️ Notification Texts Editor

Every notification text in one place — the 📱 phone push and the 🔔 in-app bell. Edit any field, then press the button at the bottom to get the SQL that saves your changes to Supabase.

How to apply changes: edit the texts → press "Copy SQL" at the bottom → paste in the Supabase SQL Editor and Run (or tell Claude "run it"). Push texts go live within ~1 minute. Bell texts apply the next time the app is opened.
Placeholders — write these inside any text and the system replaces them with the real value:
{trip} trip name {actor} who did it {item} gear item {qty} quantity {preview} update message {days} days before trip

Your edits are also saved in this browser, so you can close the page and continue later. The gray line under each pair shows a live example (trip = "Costa Rica Camp", actor = "Johnny").
