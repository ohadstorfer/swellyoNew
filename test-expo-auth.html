Expo Auth Session Test

This will test the Expo Auth Session implementation for Google OAuth.

Expected Redirect URI:

The Expo Auth Session will generate a redirect URI like:

exp://127.0.0.1:8081/--/expo-auth-session

or

https://auth.expo.io/@your-username/your-app-slug

Google Cloud Console Configuration:

You need to add the generated redirect URI to your Google OAuth client:

  1. Go to Google Cloud Console → APIs & Services → Credentials
  2. Edit your OAuth 2.0 Client ID
  3. Add the redirect URI that Expo generates

Test Links:

  • Main App

How to Test:

  1. Open the main app
  2. Click "Get Started with Google"
  3. Check the browser console for the redirect URI
  4. Add that redirect URI to Google Cloud Console
  5. Try again
