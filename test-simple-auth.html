Simple Google OAuth Test

This tests a simplified Google OAuth implementation for Expo web.

Current Configuration:

  • Client ID: ${process.env.EXPO_PUBLIC_GOOGLE_CLIENT_ID}
  • Redirect URI: http://localhost:8081 (current page origin)
  • Scopes: openid, email, profile

Google Cloud Console Setup:

Make sure your OAuth client has these settings:

  • Authorized JavaScript origins: http://localhost:8081
  • Authorized redirect URIs: http://localhost:8081
